# Ejectar en desarrollo

1. Clonar el repositorio
2. Ejecutar

```
yarn install
```

3. Tener Nest CLI instaldo
```
docker-compose up -d
```

# Stack utilizado
* MongoDB
* Nestjs
* Docker
